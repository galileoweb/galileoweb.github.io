]]> http://galileoweb.org/machtayh/2015/09/25/i-feel-connected-to-galileo-because-of/feed/ 0 Editing in Premiere Pro http://galileoweb.org/machtayh/2015/08/17/editing-in-premiere-pro/ http://galileoweb.org/machtayh/2015/08/17/editing-in-premiere-pro/#comments Mon, 17 Aug 2015 03:55:37 +0000 http://galileoweb.org/machtayh/?p=1570 When editing video in Premiere Pro begin by creating Folders in Workspace on a computer. (Certain computers will be assigned to 5th period, and others to 6th.)

Create a Project Folder with name of project (no spaces, no special characters). Make the name clear (e.g., foodfestNov15).

Inside this Folder create 4 Folders:

  • Project
  • Video
  • Audio
  • Stills

Then set the User Preferences for Premiere Pro so that all the cache files, etc, are directed the correct Folder inside Workspace.

DO THIS EVERY TIME! DO THIS EVERY TIME! DO THIS EVERY TIME! PLEASE PLEASE DO THIS EVERY TIME! Don’t get sloppy about it.

You need to move all video into your newly-created “Video” Folder in your Project Folder. Please do this! If you drag your video clips directly into Premiere Pro it will work at first. But when you disconnect the device your video content will disappear.

TO UPLOAD VIDEO

  • When you attach the camera to the computer, iPhoto will open. Import your clips into iPhoto. Select ONLY your clips (make sure you’re not importing everything on the camera).
  • Then drag the video clips into the Video folder that’s inside your Project folder.
  • Then, in Premiere Pro, drag the clips from that Video folder into the Media Browser.
